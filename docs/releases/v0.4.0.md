# Agent Harness v0.4.0

Agent Harness `0.4.0` makes the project easier to understand from a pinned
GitHub profile and hardens the visible protocol surface.

## Highlights

- Added the project-neutral [Agent Harness capability matrix](../HARNESSES.md).
- Added stable `harness-rule:*` anchors for critical protocol contracts.
- Added deterministic suite routing with:
  - `npm run test:presentation`
  - `npm run test:protocol`
  - `npm run test:all`
- Added GitHub presentation assets and README first-screen badges/links.
- Documented repository topics, release notes, and social preview guidance.

## Verification

```bash
npm run test:presentation
npm run test:protocol
npm run test:smoke
npm run validate:plugin
git diff --check
```

## Delivery Notes

This release does not add plugin hooks, daemons, watchers, telemetry, paid
services, or a separate marketing website. Agent Harness remains an
adapter-driven workflow controller for inspectable coding-agent execution.
